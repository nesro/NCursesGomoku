System equirements
==================

* g++
* libncurses-dev
* doxygen (optional)
* graphviz (optional)

Usage
=====

Compile:
<pre>
make compile
</pre>

Run:
<pre>
make run
</pre>

Clean:
<pre>
make clean
</pre>

Create a Doxygen documentation:
<pre>
make doc
</pre>
